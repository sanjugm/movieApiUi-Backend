# MovieFlix Backend

Spring Boot backend for MovieFlix application.

## Tech Stack
- Java
- Spring Boot
- Spring Security + JWT
- MySQL
- JPA / Hibernate

## Features
- User authentication with JWT
- Role-based access (ADMIN / USER)
- Movie CRUD APIs
- Poster image upload
- Secure REST APIs

## Run Locally
1. Clone repo
2. Configure MySQL in application.properties
3. Run:
   mvn spring-boot:run
